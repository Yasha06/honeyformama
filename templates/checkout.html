Умови використання
1. Загальні положення

Ці умови регулюють використання сайту "Мед для МАМИ" та покупку товарів через цей сайт.

2. Товари та ціни

Всі ціни вказані в гривнях (UAH). Ми залишаємо за собою право змінювати ціни без попередження, але ціна у вже оформленому замовленні залишається незмінною.

3. Доставка та оплата

Доставка здійснюється службою "Нова Пошта". Відправка товару відбувається протягом 1-3 робочих днів після підтвердження замовлення.

4. Повернення товару

Згідно із законодавством України, продовольчі товари належної якості обміну та поверненню не підлягають. Якщо товар пошкоджено при транспортуванні, будь ласка, складіть акт у відділенні пошти.

Політика конфіденційності
1. Збір даних

Ми збираємо лише ті дані, які необхідні для виконання замовлення: Ім'я, Телефон, Email та дані для доставки.

2. Використання даних

Ваші дані використовуються виключно для обробки замовлення та зв'язку з вами. Ми не передаємо ваші дані третім особам, окрім служби доставки "Нова Пошта" для виконання перевезення.

3. Захист даних

Ми вживаємо всіх необхідних заходів для захисту ваших персональних даних від несанкціонованого доступу.
